# Simple Pomodoro Timer

![Pomodoro Timer Screenshot](https://github.com/IsmielAbir/Simple-Pomodoro-Timer/blob/main/promodoro.PNG)

A lightweight, easy-to-use **Pomodoro Timer** built with **Python** and **Tkinter**. This application helps boost productivity using the Pomodoro Technique – 25 minutes of focused work followed by 5-minute breaks.

## Key Features
- Clean, intuitive interface with visual timer display
- Automatic switching between work (25 min) and break (5 min) sessions
- Session counter to track completed Pomodoro cycles
- Start, pause, and reset controls for flexibility
- Desktop notifications when sessions complete
